# The Athletic Club — App Concept

App mobile-first estático para hospedar na Vercel.

## Arquivos

- `index.html` — estrutura do app
- `style.css` — visual Apple-like / liquid glass
- `script.js` — navegação, tema, timers, checks e progresso
- `vercel.json` — configuração simples da Vercel

## Como publicar

1. Crie um repositório no GitHub.
2. Faça upload destes arquivos na raiz do repositório.
3. Entre na Vercel.
4. Clique em **Add New Project**.
5. Importe o repositório.
6. Framework preset: **Other** ou **Static**.
7. Deploy.

Não precisa de build command.
